## Description

Decommission a server from use including removal from backup, systems management and monitoring systems and disposal of hardware

## Resolution Notes

Completed successfully

## Implementation Plan

--Inform local IT of the necessary steps to change DHCP and static IP clients to use a different DC/DNS servers for name resolution
--Use Network Monitor to check if any DNS queries are being received by the DC before demoting, reconfigure any requestors DNS before continuing
--Notify the Operations Teams about the demotion via email
--Place the DC in outage mode in monitoring tools
--Run the AD DC Demotion Preparation tool in management tools
--Logon to the DC to be removed from Active Directory--Run DCPROMO to uninstall AD and reboot
--Uninstall DNS
--Remove the server from the domain by joining it to a workgroup and reboot
--Logon to a regional hub DC in the same domain as the DC that was removed
--Delete the computer object
--Delete any leftover DNS records
--Locate the DNS zone for the domain and then right-click it and select properties
--Click the name servers tab
--Locate and delete the entry for the removed DNS server (if present)
--Delete the server object in AD Sites and Services
--Logon to a forest root DC
--launch the DNS console on the forest root DC
--locate and delete all DNS records that point to the removed DC in the _msdcs zone and in any child zones of it 
--Also verify that no record exists in the delegated zone for the child domain zone
--Create requests to remove device from anti-virus, endpoint management, monitoring and ServiceNow CMDB

## Backout Plan

Due to the nature of this procedure any means to back out of this after the process has begun should not be implemented. Any type of recovery from backup or any other tool could cause corruption to the Active Directory database and could cause more harm than good. Once the implementation plan has been started it will need to be carried out completely.

## Test Plan

--No client issues result from the removal of this DC
--All references to this DC have been removed from Active Directory
--All references to this DC have been removed from DNS
--All references to this DC have been removed from anti-virus, endpoint management, monitoring and ServiceNow CMDB
