## Description

<p><span style="font-size: 18pt;"><strong>Excel Functionality</strong></span></p>
<p><span style="font-size: 12pt;"><strong>Copying formulas without changing cell locations</strong></span></p>
<ol><li><span style="font-size: 10pt;">When you want to know how to copy a formula and keep portions of the formula from changing relative to where they are being copied, p</span><span style="font-size: 10pt;">recede each portion of the cell specifications that aren&#39;t supposed to change with $.</span></li><li><span style="font-size: 10pt;">For example, if a formula contains the cell reference: B2, which is not supposed to change if the formula is copied to another cell, then change B2 to $B$2</span></li><li><span style="font-size: 10pt;">If, for example, the column specification, B, can change in B2, but not the row specification, 2, then change B2 to B$2 Similarly, if the row specification, 2, can change in B2, but not the column specification, B, then change B2 to $B2<br /><br /></span></li></ol>
<p><strong><span style="font-size: 12pt;">How to use formula to manipulate data in Col A and put results in Col B</span></strong></p>
<ol><li><span style="font-size: 10pt;">Type a value in box A1</span></li><li><span style="font-size: 10pt;">Click box B1</span></li><li><span style="font-size: 10pt;">Click the Summation (sigma) notation symbol in to toolbar (autosum)</span></li><li><span style="font-size: 10pt;">In the formula box, type &#34;*1.03&#34; after the &#34;&#43;SUM(A1)&#34;</span></li><li><span style="font-size: 10pt;">Hit Enter -- box B1 will now show the new value.</span></li><li><span style="font-size: 10pt;">Type in more values in col A (A2,A3,A4, etc).</span></li><li><span style="font-size: 10pt;">Highlight column B, and select Edit--&gt;Fill--&gt;FillDown, to apply </span>the formula to the entire column.</li><li><span style="font-size: 10pt;">To round values, click the B column and select Format--&gt;Style--&gt;Currency.</span></li></ol>
<div><span style="font-size: 12pt;"><strong> </strong></span></div>
<div><span style="font-size: 12pt;"><strong>Spreadsheets display 4 digit years instead of 2 digit years</strong></span></div>
<div id="solution"><ol><li><span style="font-size: 10pt;">Click on the Start Button.</span></li><li><span style="font-size: 10pt;">Pull up to Settings.</span></li><li><span style="font-size: 10pt;">Select Control Panels.</span></li><li><span style="font-size: 10pt;">Double-click on the Regional Settings Control Panel.</span></li><li><span style="font-size: 10pt;">Click on the Date tab.</span></li><li><span style="font-size: 10pt;">Change the short date style to MM/dd/yy (from MM/dd/yyyy) to return to al 2-digit format for the year.</span></li><li><span style="font-size: 10pt;">Last Updated: March 1, 2000</span></li></ol></div>
<p><strong><span style="font-size: 12pt;"> </span></strong></p>
<p><strong><span style="font-size: 12pt;">Enlarging Default Font Sizes in MS Excel</span></strong></p>
<p> Click <b>Excel</b> in the upper left hand corner.</p>
<ol><li>Select <b>Preferences...</b>.</li><li>Select the <b>General</b> tab.</li><li>Next to “Standard Font,” you can select the default font type and size.</li><li>After you click <b>OK</b>, quit Excel to apply the changes.</li></ol>
<p><strong><span style="font-size: 12pt;"> </span></strong></p>
<p><strong><span style="font-size: 12pt;">Defining the print area so that document fits on one page</span></strong></p>
<p><span>If you are unable to print the desired area of the spreadsheet on a single page, t</span><span>he print area may be defined incorrectly, or it may need to be specified.</span><br /><br /><span>Click and drag the mouse to draw a selection box around the cells that should be printed. In the file menu, highlight &#34;Print Area&#34; and select &#34;Set Print Area.&#34; This will surround the cells with a dashed line. Cells that fall outside the print area will be moved to the next page.</span><br /><br /><span>To clear an existing print area definition, highlight &#34;Print Area&#34; in the file menu, and select &#34;Clear Print Area.&#34;</span><br /><br /><span>The maximum size of the user-defined print area depends on the margins of the page. To change the margins, select &#34;Page Setup&#34; in the file menu, and click on the &#34;Margins&#34; tab.</span><br /><br /><span>To fit a large area onto a single page, it may be necessary to adjust the scaling of the printed output. To change the scaling, select &#34;Page Setup&#34; in the file menu, and click on the &#34;Page&#34; tab. The output can be scaled by an arbitrary factor, or it can be automatically fit to one or more pages.</span></p>
<p><strong><span style="font-size: 12pt;"> </span></strong></p>
<p><strong><span style="font-size: 12pt;">I&#39;m getting an error when trying to open an Excel file.</span></strong></p>
<p>Some users may get errors when opening files with macros in Excel X for Macintosh. Those whose User folders are being stored on a volume other than the startup volume are most likely to be affected. This has been affecting some people trying to submit the Excel-based academic HR forms, which are Excel-based. Deleting the preference file &#34;Office Registration Cache X&#34; often fixes the problem:</p>
<ol><li>Quit any programs you have running, especially any parts of the Office suite (Excel, Word, etc.)</li><li>Go to <b>Go &gt; Home &gt; Library &gt; Preferences &gt; Microsoft</b>.</li><li>Locate the file <b>Office Registration Cache X</b> and drag it to the trash. (<b>Note:</b> Don&#39;t empty the trash just yet.)</li><li>Try opening the Excel form again.<br />If it still doesn&#39;t work, repeat steps 1-4 and then continue:</li><li>Open up TextEdit (found under <b>Go &gt; Applications</b>) and create a new document by pressing <b>&lt;command&gt; &#43;n</b>.</li><li>Go to <b>File &gt; Save As...</b>.</li><li>Name the file <b>Office Registration Cache X</b>. (<b>Note:</b> Be sure to get the spelling and capitalization exactly.)</li><li>Save it under <b>Home &gt; Library &gt; Preferences &gt; Microsoft</b>.<br /><b>Note:</b> There are actually two libraries: one under the user&#39;s Home and one under the Macintosh HD. Be sure to use the one under Home. The exact name of user&#39;s Home is the short name of the user&#39;s local account on their Macintosh, and it will have a house icon. You can select home by pressing <b>command - shift -h</b> in the <b>Save As</b> dialogue box.</li><li>Quit TextEdit.</li><li>Go back to <b>Go &gt; Home &gt; Library &gt; Preferences</b>.</li><li>Click once on <b>Office Registration Cache X</b> to select it, and then go to <b>File &gt; Get Info</b>.</li><li>Check the box for <b>Locked</b>.</li><li>Try the file again. </li></ol>
