## Description

<p><strong><span style="font-size: 18pt;"> About Windows Vista</span></strong></p>
<p>Note: If you are migrating from an older operating system, UITS strongly recommends installing Windows 7 rather than Vista. Windows 7 was released three years after Vista and comes with more advanced features and many bug fixes, but has the same hardware requirements. Should you choose to install Vista, use the Windows Vista Upgrade Advisor to make sure your computer is capable of running Vista.</p>
<h2><span style="font-size: 12pt;">System requirements</span></h2>
<p>Microsoft&#39;s system requirements for Vista are listed below.</p>
<p>Microsoft lists Windows Vista hardware requirements in two categories, Vista Capable and Vista Premium Ready. Vista Capable computers are able to run Vista, but may have limitations in the functionality of some features. Vista Capable minimum hardware requirements are as follows:</p>
<ul><li>A modern processor (at least 800 MHz)</li><li>512 MB of system memory</li><li>A graphics processor that is DirectX 9 capable</li><li>20 GB of hard drive capacity with 15 GB free space</li><li>CD-ROM drive</li></ul>
<p>Vista Premium Ready computers can take full advantage of the features available in Vista, including the Windows Aero advanced graphical interface. Microsoft lists the minimum hardware requirements for Vista Premium as follows:</p>
<ul><li>1 GHz 32-bit (x86) or 64-bit (x64) processor <br />UITS recommends: A dual-core (Core 2) processor </li><li>1 GB of system memory <br />UITS recommends: 2 GB system memory </li><li>A graphics processor that is DirectX 9 capable with Hardware Pixel Shader v2.0 and WDDM Driver support </li><li>128 MB of graphics memory <br />UITS recommends: 256 MB graphics memory; non-integrated video card </li><li>40 GB of hard drive capacity with 15 GB free space <br />UITS recommends: 80-100 GB capacity </li><li>DVD-ROM drive </li><li>Audio output capability </li><li>Internet access capability</li></ul>
<h2><span style="font-size: 12pt;">Vista versions</span></h2>
<p>If you&#39;re upgrading from Vista Home Premium, you need to upgrade to Vista Ultimate (notVista Business) if you want to keep Windows Media Center. For information about upgrading from an earlier version of Windows, see Microsoft&#39;s <a title="" href="http://windows.microsoft.com/en-US/windows-vista/Installing-and-reinstalling-Windows-Vista#section_1" rel="nofollow">Installing and reinstalling Windows Vista</a>.</p>
