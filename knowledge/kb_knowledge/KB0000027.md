## Description

<p><strong><span style="font-size: 18pt;">Outlook 2010 - Importing Address Book From CSV File </span></strong></p>
<div id="doc-summary">
<p>This article explains how to import your address book from a CSV file into Outlook 2010. </p>
</div>
<div id="doc-body"><ol><li>Start Outlook 2010.</li><li>Select the <strong>File</strong> menu.</li><li>Choose <strong>Open.</strong></li><li>Click <strong>Import</strong>.</li><li>Select <strong>Import from another program or file</strong>, then <strong>Next</strong> from the Import/Export Wizard.</li><li>Select <strong>Comma Separated Values (Windows) </strong>, then <strong>Next</strong>. </li><li>Choose <strong>Browse </strong>to navigate to the .csv file that you want to import, and then click <strong>Open</strong> to return to the Import a File window.</li><li>Select <strong>Next</strong> and choose <strong>Contacts </strong>as the destination folder, and click <strong>Next </strong>again.</li><li>Click <strong>Finish</strong> to import the Address Book.</li></ol></div>
