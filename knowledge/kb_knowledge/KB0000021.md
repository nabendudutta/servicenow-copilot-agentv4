## Description

<p><strong><span style="font-size: 18pt;">About Windows 7</span></strong></p>
<p>Windows 7 is not an upgrade to Windows Vista, but an entirely different operating system. With it, you can expect greater functionality in communication, multimedia, and security, including improved, streamlined User Account Control (UAC) performance with fewer prompts. The interface introduced with Vista, Windows Aero, uses 3D graphics and other high-end visual effects.</p>
<p><strong><span style="font-size: 12pt;">System requirements</span></strong></p>
<p>The minimum hardware requirements for Windows 7 as specified by Microsoft are listed as follows:</p>
<ul><li>1 GHz 32-bit (x86) or 64-bit (x64) processor <br />UITS recommends: A dual-core (Core 2) processor <br /><br /></li><li>1 GB of system memory (32-bit) or 2 GB of system memory (64-bit) <br />UITS recommends: 2 GB system memory <br /><br /></li><li>A graphics processor that is DirectX 9-capable with Hardware Pixel Shader v2.0 and WDDM Driver support<br /><br /></li><li>128 MB of graphics memory <br />UITS recommends: 256 MB graphics memory; non-integrated video card <br /><br /></li><li>40 GB of hard drive capacity with 16 GB free space (32-bit) or 20 GB free space (64-bit) <br />UITS recommends: 80-100 GB capacity <br /><br /></li><li>DVD-ROM drive<br /><br /></li><li>Audio output capability<br /><br /></li><li>Internet access capability </li></ul>
<h2><span style="font-size: 12pt;">Windows 7 versions and upgrades</span></h2>
<p>The different versions and upgrades include Windows 7 Ultimate; IU departments will use Windows 7 Enterprise.</p>
<ul><li>If you are using any version of Windows 7 or Vista except Enterprise, you can upgrade to Windows 7 Ultimate. For example, if you purchase a new computer from an outside retailer, it might be shipped with Windows 7 Home Premium; you can upgrade to Windows 7 Ultimate if you wish. <br /><br /></li><li>Departments that use Vista Enterprise can upgrade to Windows 7 Enterprise. If you have Vista Enterprise for personal use, you cannot upgrade that to Windows 7 Enterprise, but you can do a clean install of Windows 7 Ultimate, which will format your hard drive and remove your previous version of Windows. <br /><br /></li><li>For all other versions of Windows, migrating to Windows 7 will require a clean install. <br /><br /></li><li>Your upgrade path must be consistent with your system type (i.e., you cannot upgrade from Windows Vista Ultimate 32-bit to Windows 7 Ultimate 64-bit, or from a 64-bit to a 32-bit operating system). </li></ul>
