## Description

<p><strong><span style="font-size: 18pt;">How to set or change your default web browser</span></strong><span class="Apple-tab-span"> </span></p>
<p>The following setting can be changed only if you have Administrator privileges on your computer.</p>
<p><span style="font-size: 12pt;"><b>Change the default programs used in Windows 8</b></span></p>
<ul><li>If not already in Desktop mode, enter it by tapping the Windows and the D key at the same time.</li><li>Move the mouse to the lower-right hand corner of the screen to activate the &#34;Gadget&#34; bar and select Settings (gear icon)</li><li>Select <span>Control Panel</span></li><li>Select <span>Programs</span></li><li>Select <span>Default Programs</span></li><li>Click on <span>Set your default programs</span></li><li>Locate and select the web browser you wish to set as your default from the list</li><li>With your choice selected, click on <span>Set this program as default</span></li><li>Click OK to save your settings</li></ul>
<p><span style="font-size: 12pt;"><b>Change the default programs used in Windows 7/Windows Vista</b></span><span></span></p>
<ul><li>Click on the <span>Start Button</span></li><li>Select <span>Control Panel</span></li><li>Select <span>Programs</span></li><li>Select <span>Default Programs</span></li><li>Click on <span>Set your default programs</span></li><li>Locate and select the web browser you wish to set as your default from the list</li><li>With your choice selected click on Set this program as default</li><li>Click OK to save your settings</li></ul>
<p><strong><span style="font-size: 12pt;">To make Firefox your default browser</span></strong></p>
<ul><li>Open Firefox</li><li>From the <span>Tools</span> pull-down menu select <span>Options</span></li><li>Click on the <span>Advanced</span> tab</li><li>In the area called <span>System Defaults</span>, click on the <span>Check Now</span> button.</li><li>If Firefox is not set to be your default browser, you will be given the option to make it the default.</li></ul>
